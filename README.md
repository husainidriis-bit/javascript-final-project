# NGO Connect – Donation & Volunteer Management System

## Overview

NGO Connect is a web-based management system designed to help nonprofit organizations manage volunteers and donations efficiently. The application provides a centralized platform for recording volunteer information, tracking donations, viewing statistics, and retrieving donor country information using the REST Countries API.

## Problem Statement

Many nonprofit organizations rely on spreadsheets and manual record keeping to manage volunteers and donations. This often leads to poor data organization, limited reporting capabilities, and inefficient workflows.

## Solution

NGO Connect simplifies nonprofit operations by providing:

* Volunteer registration and tracking
* Donation management
* Dashboard analytics
* Country information lookup
* Local data persistence
* Responsive user interface

## Installation

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Start Live Server.
4. Open index.html in your browser.
